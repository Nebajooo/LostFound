# 🏫 Lost & Found Platform

A modern full-stack web application for universities to help students and staff recover lost items through intelligent matching, secure verification, and community reporting.

## 📋 Table of Contents
- [Overview](#overview)
- [Features](#features)



## 🎯 Overview

Every day, hundreds of items are lost on university campuses - phones, wallets, ID cards, keys, and laptops. Traditional lost & found systems are inefficient, requiring students to physically visit multiple locations.

**This platform solves that by:**
- Centralizing all lost & found reports in one place
- Using intelligent algorithms to match lost items with found items
- Providing secure verification to prevent false claims
- Enabling real-time notifications for potential matches

## ✨ Features

### Core Features
| Feature | Description |
|---------|-------------|
| 🔐 **User Authentication** | Secure JWT-based login/register with university email |
| 📱 **Report Lost Items** | Submit detailed reports with private verification data |
| 🔍 **Report Found Items** | Quick submission form for found items |
| 🧠 **Intelligent Matching** | Weighted scoring algorithm (0-100% confidence) |
| ✅ **Claim Verification** | Multi-step verification with private details |
| 👑 **Admin Dashboard** | Review and manage pending claims |
| 📧 **Email Notifications** | Real-time alerts for matches and claim updates |
| 📱 **Responsive Design** | Works seamlessly on all devices |
